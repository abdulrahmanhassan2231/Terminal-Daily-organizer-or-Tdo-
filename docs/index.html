Why tdo

Built for developers who hate context-switching

Every feature is designed to keep you in the terminal and out of slow, bloated web apps.

⚡

Instant startup

Single static binary. No runtime, no VM. Starts in under 5ms even on slow machines.

🔒

Fully offline

Tasks live in a local JSON file. No account, no cloud, no internet connection needed. Ever.

🎨

Colored output

Priority-coded colors and clean formatting keep your task list readable at a glance.

📤

Markdown export

Export your task list to Markdown for standups, Notion paste-ins, or daily logs.

🦀

Written in Rust

Memory-safe, zero-overhead. Ships as a statically linked binary that runs anywhere.

📦

Single binary

Drop it in /usr/local/bin and done. No package manager, no dependencies at all.

Get started

Up and running in 30 seconds

Download the binary for your platform and drop it on your PATH.

# Download the binary
curl -Lo tdo https://github.com/abdulrahmanhassan2231/Terminal-Daily-organizer-or-Tdo-/releases/latest/download/tdo-aarch64-linux
# Make it executable
chmod +x tdo
# Move to PATH
sudo mv tdo /usr/local/bin/
# Start using it!
tdo add "My first task"
