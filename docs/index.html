// get started

Running in 30 seconds

Download the binary for your platform and put it on your PATH.

# 1 — download the binary
curl -Lo tdo https://github.com/abdulrahmanhassan2231/Terminal-Daily-organizer-or-Tdo-/releases/latest/download/tdo-aarch64-linux
# 2 — make it executable
chmod +x tdo
# 3 — move it onto your PATH
sudo mv tdo /usr/local/bin/
# 4 — capture your first task
tdo add "My first task"
